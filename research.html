Research

  1. Can A Computer Think Like A Chef - We aim to build large language models to generate constraints specific novel recipes using a list of ingredients as input.
  2. Turing Test For Chefs - To quantify the computer-generated recipes using language models, we are building a tool where users will rate the recipes as real or fake.
  3. Food Detection From A Platter - We aim to detect Indian food items from a traditional platter using transfer learning approaches. To facilitate the user and industry, we will create an android application to estimate the calorie content of each food item on a platter.
  4. Cuisine Transformation - We aim to transform a world cuisine from one region to another using deep learning techniques.
